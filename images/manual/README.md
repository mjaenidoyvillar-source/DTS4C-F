## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: Minimum 1280x720 pixels recommended
- **File Size**: Keep under 500KB per image if possible
- **Naming**: Use lowercase with hyphens (as listed above)
- **Content**: Capture the relevant UI elements clearly
- **Annotations**: You may add arrows or highlights to point out important elements
